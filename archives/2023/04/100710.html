成交量加权均线VWMA

other stock 2023/04/05

什么是成交量加权移动平均线？

MA 是过去 N 个收盘价的平均值。它赋予每个收盘价相同的权重。

  • 3 天 SMA = (C1 + C2 + C3) / 3

成交量加权移动平均线与此相同，只是它对每个收盘价赋予不同的权重。

而这个重量取决于那个时期的体积。例如，成交量大的一天的收盘价在日线图上具有更大的权重。

  • 3 日 VWMA = (C1*V1 + C2*V2 + C3*V3) / (V1+ V2+ V3)

例如，如果第 3 天的交易量 (V3) 较高，其收盘价 (C3) 将对计算值产生更大的影响。

如何使用 VWMA？

你可以：

  • 使用它的斜率作为趋势过滤器。
  • 将价格与其移动平均线进行比较以破译动量。
  • 将移动平均线视为支撑位或阻力位。

对于价格行为交易者来说，这些是交易大多数移动平均线（包括 SMA 和 EMA）的可靠方法。

这个指标能被用于识别和进行趋势交易。价格穿越指标可能意味着趋势扭转。

VWMA通常与其他信号和分析技术结合使用。

将 VWMA 与基准 SMA 相结合以突出显示交易量信号

SMA 不包括成交量权重。因此，通过将 VWMA 与 SMA 进行比较，您可以立即破译添加体积数据的影响。

SMA 是一个基准。为了使其成为有效的基准，请为 SMA 和 VWMA选择相同的回顾期。

通过这样做，我们确保两个移动平均线之间的唯一区别是交易量权重。

这里重要的是VWMA 和 SMA 之间的差距。它们的区别突出了体积权重的影响。

一般来说，成交量应顺势而增，逆势而减。

  • 如果 VWMA 高于 SMA，则意味着成交量在上涨日（市场收盘走高的日子）较高。
  • 如果 VWMA 低于 SMA，则表明下跌日成交量增加。
