JavaFX effect Shadow（暗影效果）

后端 java javafx 2019/08/06

本文于1339天之前发表，文中内容可能已经过时。

此效果会创建具有模糊边缘的指定节点的副本。

包javafx.scene.effect的名为Shadow的类表示棕褐色调效果。该类包含六个属性，它们是 -

  • color - 此属性为Color类型，表示阴影的颜色。
  • 模糊类型 - 此属性属于BlurType，它表示用于模糊阴影的模糊效果的类型。
  • radius - 此属性的类型为double，它表示阴影模糊内核的半径。
  • width - 此属性的类型为double，它表示阴影模糊内核的宽度。
  • height - 此属性的类型为double，它表示阴影模糊内核的高度。
  • input - 此属性的类型为Effect，它表示阴影效果的输入。
