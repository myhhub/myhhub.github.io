JavaFX Layout Stackpane（布局堆栈面板）

后端 java javafx 2019/08/10

本文于2046天之前发表，文中内容可能已经过时。

如果我们使用堆栈窗格，则节点排列在另一个上面，就像在堆栈中一样。首先添加的节点位于堆栈的底部，下一个节点位于堆栈的顶部。

包javafx.scene.layout的名为StackPane的类表示StackPane。该类包含一个名为alignment的属性。此属性表示堆栈窗格中节点的对齐方式。

除了这些，这个类还提供了一个名为**setMargin（）**的方法。此方法用于为堆栈窗格中的节点设置边距。
