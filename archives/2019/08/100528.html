JavaFX effect DropShadow（投影效果）

后端 java javafx 2019/08/06

本文于1352天之前发表，文中内容可能已经过时。

将此效果应用于节点时，将在指定节点后面创建阴影。

包javafx.scene.effect的名为DropShadow的类表示投影效果。该类包含九个属性，它们是 -

  • color - 此属性是Color类型，表示阴影的颜色。
  • 模糊类型 - 此属性的类型为 - BlurType，它表示用于模糊阴影的模糊效果的类型。
  • radius - 此属性的类型为double，它表示阴影模糊内核的半径。
  • width - 此属性的类型为double，它表示阴影模糊内核的宽度。
  • height - 此属性的类型为double，它表示阴影模糊内核的高度。
  • input - 此属性的类型为Effect，它表示阴影效果的输入。
  • spread - 此属性为double类型; 它代表了影子的传播。
  • offsetX - 此属性的类型为double，它表示x方向的阴影偏移（以像素为单位）。
  • offset - 此属性的类型为double，它表示y方向上的阴影偏移（以像素为单位）。
