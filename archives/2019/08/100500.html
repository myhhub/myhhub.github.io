JavaFX 2D Shapes QuadCurve形状四次曲线

后端 java javafx 2019/08/03

本文于1416天之前发表，文中内容可能已经过时。

在数学上，二次曲线是由二次函数描述的曲线，如-y = ax2 + bx + c。

在计算机图形学中使用贝塞尔曲线。这些参数曲线在所有尺度上都显得相当平滑。这些贝塞尔曲线基于XY平面上的点绘制。

二次曲线是XY平面中的Bezier参数曲线，它是2阶曲线。它使用三个点绘制：起点，终点和控制点，如下图所示

在JavaFX中，四次曲线由名为QuadCurve的类表示。该类属于包javafx.scene.shape。

通过实例化此类，您可以在JavaFX中创建QuadCurve节点。

该类有双数据类型的6个属性，即 -

  • startX - 曲线起点的x坐标。
  • startY - 曲线起点的y坐标。
  • controlX - 曲线控制点的x坐标。
  • controlY - 曲线控制点的y坐标。
  • endX - 曲线终点的x坐标。
  • endY - 曲线终点的y坐标。
