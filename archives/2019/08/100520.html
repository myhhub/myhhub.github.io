JavaFX effect Bloom（绽放效果）

后端 java javafx 2019/08/06

本文于1413天之前发表，文中内容可能已经过时。

在应用布隆效果时，使节点的某些部分中的像素发光。

包javafx.scene.effect的名为Bloom的类表示bloom效果。该类包含两个属性，它们是 -

  • input - 此属性的类型为Effect，它表示bloom效果的输入。
  • threshold - 此属性的类型为double; 这表示节点的像素的亮度的阈值。所有那些亮度大于等于该值的像素都会发光。阈值的范围是0.0到1.0。
