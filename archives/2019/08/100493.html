JavaFX 2D Shapes Rectangle形状矩形

后端 java javafx 2019/08/03

本文于1416天之前发表，文中内容可能已经过时。

通常，矩形是四边形多边形，其具有两对平行且并行的边，所有内角都是直角。

它由两个参数描述，即 -

  • height - 矩形的垂直长度称为高度。
  • width - 矩形的水平长度称为宽度。

在JavaFX中，Rectangle由名为Rectangle的类表示。该类属于包javafx.scene.shape。

通过实例化此类，您可以在JavaFX中创建一个Rectangle节点。

该类具有double数据类型的4个属性，即 -

  • X - 矩形的起点（左上角）的x坐标。
  • Y - 矩形的起点（左上角）的y坐标。
  • 宽度 - 矩形的宽度。
  • height - 矩形的高度。
