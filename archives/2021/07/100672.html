龙芯硬件平台安装Loongnix操作系统

运维 linux loongson 2021/07/20

本文于636天之前发表，文中内容可能已经过时。

​ Loongnix操作系统是龙芯开源社区推出的Linux操作系统，作为龙芯软件生态建设的成果验证和展示环境，集成了内核、驱动、图形环境等操作系统基础设施方面的最新研发成果，以“源码开放、免费下载”的形式进行发布，可直接应用于日常办公、生产、生活等应用环境，同时可供合作厂商、科研机构及爱好者等在龙芯平台上研发其品牌软件或专用系统。

龙芯硬件平台安装Loongnix系统时，对应的关系是：

操作系统版本 硬件平台
Loongnix-1.0 3A3000、3A4000 mips64处理器
Loongnix-20.mips64el 3A3000、3A4000 mips64处理器
Loongnix-20.loongarch64 3A5000 loongarch64处理器

一. 安装方式

1. DVD安装

    如果您有高速网络和 DVD 刻录机，可以选择从 Loongnix 镜像站点 下载Loongnix系统镜像，然后使用Windows/Linux系统刻录工具刻录镜像（推荐Loongnix自带刻录工具brasero）。

2. U盘安装

    因为大多数计算机能够从 USB 设备启动，所以你也可以从 USB 盘（小闪存盘）安装Loongnix。

    如果您有高速网络和大于4G的U盘，可以选择从 Loongnix镜像站点下载系统镜像，然后使用Windows/Linux制作镜像工具制作，Loongnix系统可启动终端，使用dd命令进行镜像制作。

命令：dd if=镜像文件地址 of=/dev下识别的U盘设备 bs=8M && sync

二. 安装步骤

1. 引导和开始安装

一旦 BIOS 开始从 U盘或者DVD-ROM 引导，就出现引导程序界面。这时， Linux 内核还没有加载，菜单让你选择内核开始引导，同时可以加上参数以满足你的需求。

2. 选择安装器

通过上一步选择内核引导，等待进入操作系统，此时，点击 Install Loongnix-Desktop安装程序，开启安装之旅。

3. 选择语言

安装程序已经为您检测并选好符合您当前环境的安装器语言，您也可切换成其他语言，选好语言后，点击下一步即可。

4. 选择地区、系统语言

选择地区、系统语言、数字和日期地域，支持直接在地图中点击区域，同时安装程序已经为您检测并选好符合当前环境的设置，您也可切换成您想要更改设置。选好后，点击下一步。

5. 选择键盘布局

选择键盘型号，安装器已经为您检测出适合你当前环境的键盘类型，建议使用安装器检测的配置，直接点击下一步。

6. 开始分区的工具

选择分区，安装器提供四种安装系统方式，抹除磁盘安装(新手推荐)、并存安装、取代一个分区安装、手动分区。 下图为选择默认分区格式和默认文件系统安装（新手推荐），选好后点击下一步。

默认分区格式：

分区 文件系统
/boot ext3
/ xfs
交换分区 swap

手动分区，可按照您想要的分区方式进行分区，需要注意的是，第一个分区必须先分/boot分区，文件系统类型建议分ext系列，分区完成后点击下一步。

例如：

分区 文件系统
/boot ext3（300MiB)
/ xfs（100GiB)
交换分区 swap（8.8GiB)
扩展分区 /opt ext4（300GiB)
/home/myh ext4（其它空间)

7. 新建第一个用户

新建一个用户用来登录操作系统，设置用户名好密码后，点击下一步。

8. 安装基本系统

您在设置完所有的配置选项之后的一个确认页面，上面清楚的显示出您之前设置的配置项，如无误后点击安装，如需要修改，直接点击后退即可。

9. 完成安装并重启

下图为安装完成界面，如需要直接使用新系统即可勾选现在重启，然后点击完成即可。反之，取消勾选现在重启按钮，点击完成。
